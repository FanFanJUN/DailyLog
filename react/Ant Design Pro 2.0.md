# Ant Design Pro 2.0
###### 主要改进和改变详看[官网](https://pro.ant.design/)

###### 安装
- $ git clone --depth=1 https://github.com/ant-design/ant-design-pro.git my-project
- $ cd my-project
- $ npm install
- $ npm start

使用cnpm加速
- $ npm install -g cnpm --registry=https://registry.npm.taobao.org
